# Final Stock Selection

The graded stock set is frozen for the final run:

- `steady_large_cap`: `WMT`
- `volatile_momentum`: `NVDA`
- `recent_decliner`: `UNH`
- `sideways`: `PG`

Short rationale:

- `WMT` is intended to represent a steadier large-cap name with comparatively stable trading behavior.
- `NVDA` is intended to represent a more momentum-driven and volatile stock where trend signals are likely to matter.
- `UNH` is intended to represent a recent decliner so the value-contrarian lens has a meaningful test case.
- `PG` is intended to represent a more sideways profile with less dramatic directional movement.

This stock set is frozen for the graded deliverable so outputs can be reproduced consistently.
